## 一、教学智能体名称

**码上成长 — Python AI 智能学习平台**

> 面向中职学生的 Python 编程智能辅导系统

---

## 二、智能体简介

### 2.1 解决的问题或教学痛点

| 序号 | 痛点描述 | 具体表现 | 影响分析 |
| :--- | :--- | :--- | :--- |
| 1 | 人工批改效率低下 | 教师需逐份批改学生代码作业，平均每份耗时5-10分钟 | 班级40人作业批改需3-6小时，反馈滞后1-2天 |
| 2 | 评价标准不统一 | 人工评分受主观因素影响，同一代码不同教师评分差异可达15分以上 | 学生对评价公正性存疑，影响学习积极性 |
| 3 | 个性化指导缺失 | 教师无法针对每位学生提供差异化学习建议 | 学习困难学生得不到及时帮助，优秀学生缺乏挑战 |
| 4 | 代码运行环境受限 | 学生课后练习需自行搭建Python环境，存在技术门槛 | 部分学生因环境问题无法完成课后练习 |
| 5 | 学习资源分散 | 学习资料散落在不同平台，缺乏系统化整合 | 学生寻找资源耗时，学习效率低下 |

#### 核心功能详解

##### （1）AI 智能代码评价模块
**功能定位**：自动对学生提交的 Python 代码进行多维度评价

**评价维度**：
| 维度 | 满分 | 评价要点 | 权重 |
| :--- | :--- | :--- | :--- |
| 题目理解与实现 | 30分 | 代码是否完整实现题目要求 | 30% |
| 逻辑思路 | 25分 | 条件判断、循环结构使用是否合理 | 25% |
| 代码可读性 | 25分 | 命名规范、缩进统一、结构清晰 | 25% |
| 语法掌握 | 20分 | 冒号、括号、引号配对等语法正确性 | 20% |

**界面展示**（代码评价页面）：
- 左侧：代码编辑区 + 运行按钮 + 运行结果展示
- 右侧：评价结果展示（四个维度得分条形图 + 评价评语 + 易错知识点 + 提示建议 + 类似练习题）

**技术实现**：
- 构建专业教师角色提示词（约3000字）
- 包含166项易错知识点库，供AI精准匹配
- 调用DeepSeek API进行代码分析
- 返回结构化JSON评价结果

##### （2）在线代码运行模块
**功能定位**：提供浏览器端Python代码执行环境

**技术特点**：
- 使用 `spawnSync` 执行代码，避免shell注入风险
- 设置10秒执行超时，防止无限循环
- 自动检测Python路径（兼容python3/python/不同服务器环境）
- 支持交互式输入（input()函数）
- 多编码输出处理（UTF-8/GBK/GB2312）

**界面展示**（代码编辑区）：
- 语法高亮代码编辑器
- 输入框（用于input()函数输入）
- 输出区域（显示运行结果）
- 运行/重置按钮

##### （3）AI 对话解惑模块
**功能定位**：提供学生与AI的对话式交流能力

**核心特性**：
- 多会话管理（创建、重命名、删除）
- 消息持久化存储
- 启发式教学（不直接给答案，引导思考）
- 严格限制只回答Python相关问题
- 响应式布局（移动端底部浮动按钮）

**界面展示**（AI解惑页面）：
- 左侧：会话列表侧边栏
- 右侧：消息对话区 + 输入框
- 底部：提示词模板快捷按钮

##### （4）学情统计分析模块
**功能定位**：展示学生学习数据和班级统计信息

**学生端功能**：
- 个人评价历史记录
- 成绩趋势图表
- AI生成的个性化分析报告

**教师端功能**：
- 班级平均成绩统计
- 各等级人数分布
- 知识点掌握频次统计
- Excel导出功能

**界面展示**（学情统计页面）：
- 统计卡片（总提交数、平均分、优秀率等）
- 图表区域（柱状图、折线图）
- 评价记录列表

##### （5）题库管理模块
**功能定位**：支持教师维护Python练习题

**数据结构**：
- 题目ID（自增主键）
- 题目标题
- 题目内容
- 提示信息
- 示例代码

**界面展示**（题库管理页面）：
- 题目列表（表格形式）
- 新增/编辑/删除操作按钮
- 搜索筛选功能

##### （6）学习资源管理模块
**功能定位**：管理平台推荐的外部学习资源链接

**数据结构**：
- 资源ID（自增主键）
- 资源标题
- 资源URL
- 资源描述
- 排序序号

**界面展示**（学习资源页面）：
- 学生端：资源导航标签 + iframe内嵌展示（高度800px）
- 教师端：资源列表 + 增删改操作

- 教师布置编程练习题

- 实时获得AI评价反馈

**场景二：课后作业**
- 学生课后完成作业提交

- 学生根据评价报告自主改进
- 教师查看班级整体情况

**场景三：个性化辅导**
- AI生成个性化分析报告
- 教师根据报告进行针对性辅导
- 学生通过AI解惑解决疑难问题
