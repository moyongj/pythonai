第一章  软件概述
============================================================

"码上成长Python智能学习平台"是一款面向中职学生的Python编程智能学习软件。
软件基于Web浏览器运行，采用前后端一体化架构，集成了AI大模型能力，
为学生提供智能代码评价、个性化学习建议、AI对话解惑等核心功能，
同时为教师（管理员）提供学生管理、题库维护、学情统计等教学辅助功能。

【软件的主要功能包括】

1. 学生端功能
   （1）试题演练：浏览和练习平台内置的Python编程题目；
   （2）AI代码评价：提交Python代码，AI自动进行多维度智能评价；
   （3）学情统计：查看个人历史评价记录、成绩趋势和知识点掌握情况；
   （4）AI解惑对话：与AI智能助手进行对话，获得编程问题引导；
   （5）学习资源：浏览平台推荐的Python学习资源链接。

2. 管理员端功能
   （1）学生管理：学生信息的增删改查、批量导入、密码管理；
   （2）题库管理：Python练习题目的新增、编辑、删除和查看；
   （3）学情记录：查看所有学生的代码评价历史记录；
   （4）学情分析：班级维度的统计图表、知识点分布分析；
   （5）学习资源管理：管理平台推荐的外链学习资源。

【软件的技术特点】

1. AI智能评价：集成OpenAI兼容接口，对学生Python代码进行语法正确性、
   知识点匹配、代码风格等多维度智能分析；
2. 在线代码运行：内置Python代码执行环境，支持交互式输入/输出；
3. 多会话AI对话：支持创建多个对话主题，消息持久化存储；
4. 数据可视化：使用Recharts图表库展示学情统计数据；
5. 响应式界面：基于Tailwind CSS，适配桌面端和移动端。

第二章  运行环境
============================================================

【硬件环境要求】
  - CPU：双核及以上
  - 内存：4GB及以上
  - 硬盘：500MB可用空间（不含数据库增长空间）
  - 网络：宽带互联网连接（用于AI服务调用）

【软件环境要求】
  - 操作系统：Windows 10及以上 / macOS 10.15及以上 / Linux发行版
  - Node.js：版本18.17.0及以上（推荐20.x）
  - pnpm：版本9.0.0及以上
  - Python：3.8及以上（用于在线代码运行功能）
  - 浏览器：Chrome 90+ / Edge 90+ / Firefox 88+ / Safari 14+
  - 数据库：SQLite 3.x（软件内置，无需独立安装）

【网络环境要求】
  - 需要能够访问AI服务API（支持OpenAI API及兼容接口如ChatGLM）
  - 若在内网环境使用，需配置AI服务的公网访问或本地化部署模型

第三章  安装与启动
============================================================

【第一步：安装依赖】

1. 将软件部署包解压到目标目录；
2. 打开系统命令行（终端），进入软件目录；
3. 执行以下命令安装依赖：

    pnpm install

  （注意：必须使用pnpm，不可用npm或yarn）

【第二步：配置环境变量】

1. 在软件根目录找到".env"文件（若无则新建）；
2. 按以下说明填写配置项：

  | 配置项            | 说明                                  | 示例                    |
  |-------------------|---------------------------------------|-------------------------|
  | OPENAI_API_KEY   | AI服务API密钥                        | sk-xxxxxxxx              |
  | OPENAI_BASE_URL  | AI服务接口地址（可选）                | https://api.openai.com/v1|
  | OPENAI_MODEL     | 使用的AI模型名称（可选）              | gpt-4o-mini             |
  | HOSTNAME         | 服务监听地址（可选，默认0.0.0.0）   | 0.0.0.0                |
  | PORT             | 服务监听端口（可选，默认3000）        | 3000                    |

【第三步：启动软件】

开发模式启动（推荐用于测试）：
  执行命令：pnpm run dev
  启动后访问：http://localhost:3000

生产模式启动：
  ① 构建：pnpm run build
  ② 启动：pnpm run start
  启动后访问：http://服务器IP:端口

【初始化说明】

软件首次启动时，会自动完成以下初始化工作：
1. 在 data/ 目录下创建 SQLite 数据库文件（mscz.db）；
2. 创建默认管理员账号（用户名：myj，密码：123456）；
3. 创建默认学生账号（学号：01、02、03……，密码：123456）；
4. 导入默认Python练习题库（约60道题目）；
5. 导入默认学习资源链接。

4.4  AI代码评价（核心功能）
------------------------------

【功能说明】此为软件的核心功能。学生将编写好的Python代码粘贴至
代码编辑区并提交，系统调用AI大模型对代码进行全方位智能分析，
生成多维度评价报告，包括：语法正确性、知识点掌握情况、代码风格
评价、综合评分和个性化学习建议。

【操作步骤】

1. 点击顶部导航栏的"代码评价"按钮；
2. 在"选择题目"下拉框中选择要练习的题目（必选）；
3. 在代码编辑区（中央大型文本输入框）中输入或粘贴Python代码；
4. 点击"提交评价"按钮；
5. 等待AI分析（通常3~10秒，视AI服务响应速度而定）；
6. 页面展示AI评价报告，包括以下内容：

   （1）评价维度分项展示：
       - 题目理解与实现：评分及评价说明
       - 逻辑思路：评分及评价说明
       - 代码可读性：评分及评价说明
       - 语法掌握：评分及评价说明

   （2）综合评分：百分制总分，以及对应的等级（优秀/良好/及格/待提高）；

   （3）知识点识别：AI自动识别代码中涉及到的Python知识点；

   （4）改进建议：针对当前代码的个性化学习建议和提升方向。

（请在此处插入代码评价页面截图——展示代码输入区和评价报告展示区）

【代码在线运行】

在代码评价页面，学生还可以：
1. 点击"运行代码"按钮；
2. 若代码需要输入（使用input()函数），在弹出的输入框中填写；
3. 系统执行Python代码并展示运行结果（输出/错误）；
4. 运行超时时间设为10秒，防止死循环导致服务器阻塞。

【历史记录】

每次评价完成后，记录自动保存。学生可随时在"学情统计"页面查看
所有历史评价记录，不需要重复提交即可回顾AI评价结果。

------------------------------

【功能说明】学生可在此页面查看个人的学习档案，包括历史评价记录、

【操作步骤】

1. 点击顶部导航栏的"学情统计"按钮；
2. 页面展示以下内容：

       - 总提交次数

（请在此处插入学情统计页面截图——展示统计卡片、趋势图、知识点分布图）

【生成学情分析报告】

  - 后续学习规划（阶段性学习目标）

报告以美观的HTML页面展示，支持中文排版。

4.6  AI解惑对话
------------------------------

【功能说明】学生可与AI智能助手进行对话式交流，询问Python编程问题，
获得启发式引导和提示（而非直接给出完整答案），培养独立解决问题的能力。

【操作步骤】

1. 点击顶部导航栏的"AI解惑"按钮；
2. 页面左侧为"会话列表"：
   - 点击"新建对话"可创建新的对话主题；
   - 点击已有会话可切换至该对话；
   - 支持重命名会话、删除会话；
3. 页面右侧为对话区域：
   - 在底部输入框中输入问题；
   - 按"发送"按钮或按Ctrl+Enter发送；
   - AI回复会实时显示在对话区域；
4. 对话内容支持Markdown格式渲染（代码高亮、标题、列表等）；

（请在此处插入AI对话页面截图——展示会话列表和对话区域）

【AI对话规则】

  - AI助手严格限制只回答Python相关的编程问题；
  - 对于非Python问题，AI会礼貌拒绝并引导学生提出Python相关问题；
  - AI采用启发式教学，不直接给出完整代码答案，而是通过提示和引导
    帮助学生自己找到解决方案；
  - 所有对话消息均保存在数据库中，刷新页面或重新登录后历史消息不丢失。

【移动端适配】

在移动端设备上，会话列表隐藏为侧边栏，点击左上角按钮可展开/
收起会话列表，操作体验与桌面端一致。

第六章  常见问题与注意事项
============================================================

【常见问题】

Q1：学生忘记密码怎么办？
A：由管理员在"学生管理"模块中对该学生执行"重置密码"操作，
   密码将重置为默认密码（123456），学生登录后如需修改密码
   可联系管理员再次重置。

Q2：AI评价一直没有响应怎么办？
A：请检查以下几点：
   1. 确认.env文件中OPENAI_API_KEY配置正确；
   2. 确认服务器能够访问AI服务接口地址；
   3. 检查AI服务账户是否有足够余额；
   4. 若AI服务响应超时，可尝试更换模型或切换AI服务供应商。

Q3：在线运行Python代码时报错"未找到Python环境"怎么办？
A：请在服务器上安装Python（3.8及以上版本），并确保python或
   python3命令可在命令行中直接执行。软件会自动检测Python路径。

Q4：如何备份数据？
A：平台的全部数据存储在 data/mscz.db 文件中，定期复制该文件
   至安全位置即可完成数据备份。

Q5：软著申请时需要提供什么材料？
A：需要以下三类材料：
   1. 软件著作权登记申请表（在中国版权保护中心官网在线填写）；
   2. 软件源代码文档（前30页+后30页，每页不少于50行）；
   3. 软件用户操作手册（本手册即为此目的撰写）。

【注意事项】

1. 本软件为教育辅助工具，AI评价结果仅供参考，不作为正式考试成绩；
2. 使用AI功能需要有效的AI服务API密钥，请妥善保管，不要泄露；
3. 建议定期备份数据库文件（data/mscz.db）；
4. 若部署在公网服务器，请务必修改默认管理员密码和默认学生密码；
5. 本软件使用的AI服务由第三方提供，服务可用性和评价质量取决于
   AI服务供应商；
6. 软件内置题库可根据教学需要随时由管理员增删改查，建议定期
   更新题目以保持教学内容与课程进度同步。
