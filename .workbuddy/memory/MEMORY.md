# 项目记忆 - 码上成长Python智能学习平台

## 项目概况
- **名称**: 码上成长Python智能学习平台
- **技术栈**: Next.js 16 (App Router) + React 19 + TypeScript 5 + shadcn/ui + Tailwind CSS 4 + SQLite + OpenAI API
- **核心功能**: AI代码评价、在线Python运行、学情统计、AI对话解惑、题库管理
- **双角色**: 学生端 + 管理员端
- **包管理**: 仅使用 pnpm
